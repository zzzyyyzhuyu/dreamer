#### 简介
- 全局异常处理，统一返回值封装
- 整合redis相关以及工具类封装
- 整合mybatis+tk通用mapper，通用biz、controller封装
- 基于OAuth2和 Spring-security的token认证授权机制，可扩展三方登录
